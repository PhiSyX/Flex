# Custom Element (natif)

Le nom d'un fichier d'un custom element DOIT être nommé en `kebab-case` qui correspond au nom de la balise de ce
dernier.

## Exemple

```bash
┌───(PhiSyX@フィジックス)
└── λ ls
    └── my-module/
        ├── package.json
        └── custom_elements/
            ├── custom-tag-name.js
            └── my-tag-name.js
```
