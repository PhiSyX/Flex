# Structure

La convention de nommage choisie pour le nom d'une structure est `PascalCase` ou
`snake_case`.

La convention de nommage choisie pour un champ de structure est `snake_case`.

## Exemple

```rs
// Rust style
struct FooBar 
{
	foo_bar: usize,
}

// C style
struct foo_bar 
{
	bar: usize,
}
```
