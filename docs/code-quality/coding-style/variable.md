# Variable

## Convention

La convention de nommage choisie pour le nom d'une variable est `snake_case`.

## Préférence

Utiliser `let` au lieu de `const` pour tout, sauf pour les valeurs constantes.

## Exemple Rust

```rs
let foo = 42;
let mut foo_bar = 42;
```

## Exemple TypeScript

```ts
let foo = 42;
let foo_bar = 42;
```
